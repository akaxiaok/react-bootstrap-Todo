# react-todo

A demo use react, bootstrap, webpack and more

## How to use
- First
``` bash
$ npm install webpack & mocha -g
$ npm install
```

- Start a server on local:8080 and open browser automatically 
```bash
$ npm start
```
- Test

```bash
$npm test
```

- Build to /build

```bash
$ npm run build 
```
